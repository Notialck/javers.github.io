Features

With version {{site.javers_current_version}} you can:

  • Compare complex object structures using javers.compare(Object oldVersion, Object currentVersion)

  • Commit changes performed on your domain objects calling javers.commit(String author, Object currentVersion)

  • Take advantage of the auto-audit annotation for Spring Data Repositories.

  • Persist data change history in MongoDB, PostgreSQL, MySQL, Oracle or Microsoft SQL Server.

  • Track every change of object fields as well as relations changes.

  • Use Javers Query Language to easily query for changes of a given class, object or property.

  • Browse data change history in two views — diffs and snapshots.

Demo app

Test it out in a few seconds

Check out our cool Demo webapp based on spring-boot. Launch it from command line using provided Gradle wrapper. Requiremets: MongoDB (running mongod at localhost).

git clone https://github.com/javers/javers-demo.git
./gradlew bootRun

In a few seconds, app should be up & ready at http://localhost:8080
